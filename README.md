# ComboPassDict - 组合密码字典生成工具

## 项目介绍

ComboPassDict 是一个简单实用的 Java Swing 桌面工具，用于快速生成密码字典（密码爆破字典）。  
它支持将三组基础词表（A、B、C 列）按照指定方案进行组合拼接，生成常见的用户名+符号+数字/年份等组合形式，适用于安全测试、渗透测试中的弱口令字典生成。

**特点：**
- 支持多种组合方案：A、B、C、AB、AC、BC、ABC
- 支持保存/加载上次配置（`config.ini`）
- 界面简洁，操作简单，一键生成

**注意：** 本工具仅用于合法的安全测试与授权渗透测试，请严格遵守法律法规，禁止用于任何非法用途。

## 界面预览

程序启动后界面如下：

- **生成方案**：填写组合方式，例如 `AB,AC,ABC`（默认值），支持用逗号或空格分隔
- **A列**：通常放用户名、常见前缀（如 admin、root、test）
- **B列**：通常放特殊符号（如 @、#、!、_ 等）
- **C列**：通常放数字、年份、常见后缀（如 123、2025、888 等）

点击 **生成字典** 即可生成并保存文件，点击 **保存方案** 可保存当前内容，下次打开自动加载。
生成的文件名为 `dic202512231530.txt`，保存在程序运行目录下。
